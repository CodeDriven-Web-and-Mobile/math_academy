## 🎨 Color Examples

### Grade Gradients
- **Grade 6**: `Blue → Cyan` (Ocean theme)
- **Grade 7**: `Purple → Pink` (Sunset theme)
- **Grade 8**: `Green → Emerald` (Nature theme)
- **Grade 9**: `Orange → Red` (Fire theme)
- **Grade 10**: `Indigo → Purple` (Royal theme)
- **Grade 11**: `Pink → Rose` (Blossom theme)
- **Grade 12**: `Teal → Cyan` (Sky theme)

### Difficulty Colors
- **Easy**: Green background, dark green text
- **Medium**: Yellow background, dark yellow text
- **Hard**: Orange background, dark orange text
- **Advanced**: Red background, dark red text

## ✨ Interactive Elements

### Hover Effects
- Cards lift up slightly (transform: translateY(-4px))
- Border color changes to blue
- Shadow increases
- Smooth transition (300ms)

### Click Effects
- Button press animation
- Color change on active state
- Ripple effect (visual feedback)

### Transitions
- Page transitions: Smooth fade
- Card animations: Slide up
- Progress bar: Smooth fill
- Score updates: Number animation
